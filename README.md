# Vinhomes Hải Vân Bay - Premium Landing Page

A high-conversion landing page for the Vinhomes Hải Vân Bay real estate project in Da Nang, Vietnam. Built with modern web technologies, focusing on premium aesthetics and responsive design.

## 🚀 Technologies Used

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Package Manager**: pnpm

## ✨ Features

- **Premium Design**: Modern aesthetic with glassmorphism, smooth gradients, and rich animations.
- **Asymmetric Interactivity**: Unique hover effects on navigation and UI elements.
- **Dynamic Header**: Fixed navigation with scroll-based transparency and scale transitions.
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Interactive Hotline**: Floating contact buttons (Zalo, Phone) with ripple effects.
- **Rich Media**: High-quality background videos and optimized image assets.
- **SEO Optimized**: Following best practices for search engine visibility.

## 🛠️ Development

### Installation

```bash
pnpm install
```

### Local Development

```bash
pnpm dev
```

### Build for Production

```bash
pnpm build
```

## 🏗️ Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable UI components (Navbar, Hero, Footer, sections, etc.).
- `public/`: Static assets (images, videos).
